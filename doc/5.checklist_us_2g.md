**US-RPT-01: Monthly Overview**
- [ ] `getMonthlyOverview(authUserId, year, month)`:
  - Resolve profileId
  - SELECT transactions WHERE userId = profileId AND date IN range
  - totalIncome = Σ amount WHERE type = 'income'
  - totalExpense = Σ amount WHERE type = 'expense'
  - transactionCount = tổng số transactions
  - incomeCount = số transactions income
  - expenseCount = số transactions expense
  - difference = totalIncome − totalExpense
  - Return: `MonthlyOverview`

- [ ] `getPreviousMonthComparison(authUserId, year, month)`:
  - Tính previousMonth = month − 1 (handle year overflow)
  - Gọi getMonthlyOverview cho month hiện tại và month trước
  - incomeChangePercent = ((currentIncome − prevIncome) / prevIncome) × 100 (hoặc null nếu prevIncome = 0)
  - expenseChangePercent = ((currentExpense − prevExpense) / prevExpense) × 100 (hoặc null nếu prevExpense = 0)
  - Return: `MonthComparison`

**US-RPT-02: Expenses by Category**
- [ ] `getExpensesByCategory(authUserId, startDate, endDate)`:
  - SELECT transactions WHERE userId = profileId AND type = 'expense' AND date IN range
  - JOIN với categories để lấy name + icon
  - Group by categoryId
  - totalAmount = Σ amount
  - transactionCount = COUNT(*)
  - percentage = (totalAmount / Σ totalAmount all categories) × 100
  - Return: `CategoryExpense[]`

- [ ] `getTransactionsByCategory(authUserId, categoryId, startDate, endDate)`:
  - SELECT transactions WHERE userId = profileId AND categoryId = param AND type = 'expense' AND date IN range
  - ORDER BY date DESC, amount DESC
  - Return: `Transaction[]`

**US-RPT-03: Income/Expense Trend**
- [ ] `getIncomeExpenseTrend(authUserId, months)`:
  - months: 3 | 6 | 12 | custom
  - Tính startMonth = currentMonth − months + 1
  - Loop qua từng tháng trong range:
    - Gọi getMonthlyOverview cho mỗi tháng
    - Lưu totalIncome, totalExpense
  - Tính averageExpense = Σ totalExpense / months
  - Return: `TrendDataPoint[]` (sorted by month ASC)

**US-RPT-04: Budget vs Actual**
- [ ] `getBudgetVsActual(authUserId, year, month)`:
  - Lấy month range từ getMonthRange
  - SELECT budgets WHERE userId = profileId AND month = `${year}-${month}`
  - SELECT expenses by category trong tháng (từ getExpensesByCategory)
  - Join budgets với expenses:
    - budgetAmount = budget.amount (cents)
    - actualAmount = Σ expense cho category đó
    - percentage = (actualAmount / budgetAmount) × 100
  - Return: `BudgetComparisonItem[]` (bao gồm cả categories không có budget)

**US-RPT-05: Net Worth History**
- [ ] Re-export `getNetWorthHistory` từ assetService (đã có từ US-AST-08)

**US-RPT-06: Asset Allocation**
- [ ] Re-export `getNetWorth` từ assetService (đã có từ US-AST-06)
- [ ] Lọc bỏ debt type, chỉ lấy assets

**US-RPT-07: Top Expenses**
- [ ] `getTopExpenses(authUserId, startDate, endDate, limit = 10)`:
  - SELECT transactions WHERE userId = profileId AND type = 'expense' AND date IN range
  - ORDER BY amount DESC
  - LIMIT param
  - JOIN với categories để lấy name + icon
  - Return: `Transaction[]` (đã sort desc)

**US-RPT-08: Annual Summary**
- [ ] `getAnnualSummary(authUserId, year)`:
  - Loop 12 tháng (01 → 12)
  - Gọi getMonthlyOverview cho mỗi tháng
  - monthly[] = [{ month: 'YYYY-MM', income, expense, difference }]
  - totalIncome = Σ income
  - totalExpense = Σ expense
  - averageIncome = totalIncome / 12
  - averageExpense = totalExpense / 12
  - Return: `AnnualSummary`

## Ghi chú quan trọng

- **⚠️ KHÔNG thay đổi DB** — sử dụng tables `transactions`, `categories`, `budgets`, `assets`, `assetHistory` đã có
- **Route order:** `/categories/:categoryId/transactions` PHẢI mount TRƯỚC `/:id` (tránh Hono match nhầm param)
- **Date handling:** Dùng `new Date().toLocaleDateString('sv')` = YYYY-MM-DD (KHÔNG dùng `toISOString()`)
- **Monetary:** Tất cả monetary values lưu/hiển thị là cents → divide by 100 trước formatCurrency
- **formatCurrency:** Dùng từ `apps/web/src/utils/format.ts`
- **Chart.js:** Dùng pattern từ Asset Tracker components (AssetPieChart, NetWorthChart)
- **Period filter:** Global state, shared giữa all tabs
- **Tab structure:** Tổng quan | Theo danh mục | Xu hướng | Ngân sách | Tài sản | Báo cáo năm
- **Drill-down:** Click category row → mở modal với transaction list
- **Empty states:** Hiển thị message khi không có data
- **Reuse:** Net Worth history và Asset Allocation từ assetService đã có
- **Budget vs Actual:** Chỉ hiển thị khi có budget data
- **Annual summary:** Tính average = total / 12
- **Trend average line:** Dashed line với giá trị trung bình expense
- **Prototype CSS classes:** `.period-filter`, `.period-btn`, `.overview-card`, `.category-table`, `.drilldown-modal`, `.chart-wrap`
