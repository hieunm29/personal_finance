### 0.4. Backend — Service (`apps/server/src/services/assetService.ts`) — tạo mới
Pattern: first param `authUserId: string`, resolve profileId nội bộ

**Metadata shapes (lưu JSON trong `assets.metadata` TEXT):**
```
'cash':        metadata = null
'bank':        { bankName: string }
'gold':        { unit: 'chi' | 'luong'; quantity: number; buyPrice: number }   // buyPrice cents/unit
'stock':       { ticker: string; quantity: number; avgBuyPrice: number }        // avgBuyPrice cents/share
'savings':     { bankName: string; depositAmount: number; interestRate: number; term: number; depositDate: string }
'real_estate': { address?: string }
'debt':        { originalAmount: number; interestRate: number; startDate: string; term: number }
```

- [x] `getAssets(authUserId, type?)` — SELECT assets WHERE userId = profileId (+ optional WHERE type), ORDER BY createdAt DESC, return Asset[]
- [x] `createAsset(authUserId, data: CreateAssetInput)` — INSERT asset, set metadata = JSON.stringify(data.metadata ?? null), return Asset
- [x] `updateAsset(authUserId, assetId, data: UpdateAssetInput)` — ownership check (asset.userId === profileId), partial UPDATE, return updated Asset
- [x] `deleteAsset(authUserId, assetId)` — ownership check, DELETE asset (cascade assetHistory)
- [x] `getNetWorth(authUserId)` — SELECT all assets, Σ(non-debt currentValue) − Σ(debt currentValue), group by type → NetWorthData
- [x] `updateAssetValue(authUserId, assetId, newValue: number)` — ownership check, UPDATE assets SET currentValue = newValue, INSERT assetHistory (assetId, value=newValue, date=today YYYY-MM-DD), return updated Asset
- [x] `getNetWorthHistory(authUserId, limit?)` — aggregate assetHistory by month: lấy snapshot mới nhất của mỗi asset trong từng tháng, tính net worth theo tháng, ORDER BY month ASC, return NetWorthHistoryPoint[]

- [x] `getNetWorth(authUserId)`:

  - SELECT tất cả assets WHERE userId = profileId

## Ghi chú quan trọng

- **⚠️ KHÔNG thay đổi DB** — tables `assets` + `assetHistory` đã tồn tại trong schema
- **Route order:** `/networth` và `/networth-history` PHẢI mount TRƯỚC `/:id` (tránh Hono match nhầm param)
- **metadata:** `JSON.stringify` khi lưu, `JSON.parse` khi đọc; null cho type 'cash'
- **Monetary:** lưu cents integer, hiển thị qua `formatCurrency(cents)` từ `apps/web/src/utils/format.ts`
- **Cache invalidation:** invalidate `['assets']` prefix sau mọi mutation (covers useAssets + useNetWorth + useNetWorthHistory)
- **Tab mapping:** Tiền mặt = ['cash','bank'], Vàng = 'gold', Cổ phiếu = 'stock', Tiết kiệm = 'savings', Nợ = 'debt', BĐS = 'real_estate'
- **Net Worth calc:** Σ(non-debt currentValue) − Σ(debt currentValue)
- **assetHistory:** INSERT khi `updateAssetValue` được gọi, date = today YYYY-MM-DD (dùng `new Date().toLocaleDateString('sv')`)
- **NetWorthHistory logic:** lấy snapshot mới nhất của mỗi asset per month từ assetHistory, aggregate
- **P&L gold:** `costBasis = quantity × buyPrice`, `pnl = currentValue − costBasis`
- **P&L stock:** `costBasis = quantity × avgBuyPrice`, `pnlPercent = pnl / costBasis × 100`
- **Savings maturity:** `new Date(depositDate).setMonth(month + term)` → maturityDate; dùng `toLocaleDateString('sv')` (KHÔNG dùng `toISOString()`)
- **Savings interest:** `depositAmount × (interestRate/100) × (term/12)`
- **Near maturity badge:** `daysUntilMaturity <= 7 && daysUntilMaturity > 0`
- **Prototype CSS:** `.asset-item`, `.asset-icon`, `.asset-info`, `.asset-name`, `.asset-sub`, `.chart-wrap`, `.tabs`, `.tab.active`
- **Dashboard:** `dashboardService.ts` trả `netWorth` thực từ `getNetWorth()`, wrapped try/catch → null nếu lỗi
