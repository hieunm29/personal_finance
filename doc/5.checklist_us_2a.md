## Phần 2 — US-AUTH-02: Đăng nhập

> AC: Email + password → JWT tokens, access 30 phút, refresh 7 ngày, redirect Dashboard.

### Backend

- [x] **Auth middleware (inline trong `apps/server/src/index.ts`)**
  - better-auth handler: `app.on(['POST', 'GET'], '/api/auth/**', (c) => auth.handler(c.req.raw))`
  - Auth middleware cho `/api/*`: `auth.api.getSession({ headers: c.req.raw.headers })`
  - Nếu hợp lệ → `c.set('userId', session.user.id)` → `next()`
  - Nếu không → trả `401 { error: { code: 'UNAUTHORIZED', message: 'Unauthorized' } }`

### Frontend

- [x] **`apps/web/src/pages/LoginPage.tsx`**
  - Form fields: email, password
  - Validation bằng `react-hook-form` + `signInSchema`
  - Submit: `authClient.signIn.email({ email, password })`
  - Thành công → navigate `/` (Dashboard)
  - Lỗi → hiển thị "Email hoặc mật khẩu không đúng" (không tiết lộ cụ thể)
  - Link "Chưa có tài khoản? Đăng ký" → navigate `/register`

### Acceptance Tests (manual)

- [x] Đăng nhập đúng → redirect Dashboard
- [x] Sai email/password → hiển thị lỗi chung
- [x] Sau đăng nhập, gọi API protected → nhận dữ liệu (cookie session hoạt động)
