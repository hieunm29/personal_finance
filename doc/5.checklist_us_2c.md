## Ghi chú quan trọng

- **isDefault:** Danh mục/nhóm mặc định (isDefault=true) → chỉ ẩn được, không xóa được
- **Transaction picker vs Category manager:** picker chỉ lấy `isVisible=true`, manager lấy tất cả
- **SQLite RETURNING:** Drizzle SQLite không hỗ trợ `.returning()` kèm JOIN → sau INSERT query lại bằng ID
- **Unique name check:** Kiểm tra phía service (không có DB unique constraint) — LOWER(name) so sánh
- **Transaction re-assign:** khi xóa category có transactions, dùng Drizzle UPDATE SET categoryId=replacement WHERE categoryId=deleted
- **sortOrder mới:** = max(sortOrder existing user groups) + 1 để đặt nhóm/danh mục mới cuối danh sách
