## Ghi chú quan trọng

- **Monetary:** Toàn bộ amount lưu cents (integer). VND: user nhập "150000" → lưu 15000000 (×100). Nhất quán với DB schema và `formatCurrency` hiện tại.
- **SQLite RETURNING:** Drizzle SQLite không hỗ trợ `.returning()` → sau INSERT phải query lại bằng ID.
- **Filter route vs /:id route:** `GET /export` phải đặt trước `GET /:id` để tránh bị match nhầm.
- **Category ownership:** Categories được seed theo userId, query phải filter theo userId.
- **Auth middleware:** Routes /api/categories, /api/wallets, /api/transactions đều cần userId từ session — mount sau auth middleware.
